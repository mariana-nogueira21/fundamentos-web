# 🌐 Fundamentos Web

Este repositório reúne os primeiros passos no desenvolvimento web, com foco em **HTML** e **CSS**. É ideal para quem está começando e quer entender como construir páginas estáticas com estrutura e estilo.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

## 📁 Estrutura do projeto

Os arquivos estão organizados por temas e exercícios que abordam:

- Estrutura básica de uma página HTML
- Elementos semânticos
- Estilização com CSS
- Layouts simples com Flexbox
- Prática de responsividade

## 📌 Objetivos

- Compreender a estrutura de documentos HTML
- Aplicar estilos com CSS de forma eficiente
- Desenvolver páginas web simples e funcionais
- Praticar boas práticas de organização de código
